# Employee Management System

A professional Employee Management System developed as part of an SDET portfolio project.

## Project Overview

This project allows users to manage employee records through a clean and responsive web interface.

Users can:

- Add Employee
- View Employee List
- Edit Employee Details
- Delete Employee
- Search Employees
- Store data using Local Storage
